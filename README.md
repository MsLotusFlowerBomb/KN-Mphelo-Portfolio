# KN Mphelo Portfolio

A static personal portfolio website for **Kgaugelo Nkosazana Mphelo** showcasing cybersecurity, software development, and AI integration work in a polished, terminal-inspired UI.

## Project structure

- `index.html` — homepage featuring the hero introduction, featured project previews, skill summaries, achievements, and contact section.
- `projects.html` — detailed project showcase with category filters and deep project descriptions.
- `resume.html` — résumé-style page with professional summary, achievements, skills, experience, education, and certifications.
- `shared.css` — shared theme and layout styling applied across all pages.

## Key highlights

- Modern dark portfolio design with neon accents and retro terminal styling.
- Responsive layout optimized for desktop and mobile devices.
- Resume page includes print/download support via browser printing.
- Project showcase includes multiple categories: cybersecurity, full stack, AI, and academic work.
- Built entirely with static HTML and CSS.

## Featured projects

- **UL Optometry Clinic Management System** — cross-platform clinical management app built with C# and .NET MAUI, using Supabase and PostgreSQL Row-Level Security.

- **TenderShield SA** — AI-powered procurement fraud detection platform with Claude API and regulation analysis for PPPFA, PFMA, MFMA, B-BBEE, CIDB, CSD, and POPIA.

- **StealthMode — Static Context Engine** — offline AI coding assistant using Gemini API, LangChain4j, and Apache Lucene.

- **NowNow Courier System** — full-stack Java EE courier platform with customer, driver, and admin portals.

## How to use

1. Open `index.html` in your browser.
2. Use the top navigation links to access `projects.html` and `resume.html`.
3. Use the download/print button on the résumé page to save a PDF copy.

## Deployment

This is a static website and can be hosted on any static hosting service.

Suggested hosting options:

- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting

## Notes

- No build tools or dependencies are required.
- The site uses a remote Google Fonts stylesheet for typography.
- All pages are fully self-contained with styling provided by `shared.css`.
